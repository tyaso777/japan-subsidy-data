# japan-subsidy-data

日本の補助金公表資料を、分析しやすいCSV・JSONL・Excel・HTMLに整理する非公式の研究用リポジトリです。

## 収録プロジェクト

- `補助金/大規模成長投資補助金`: 1次～4次の交付決定企業公開PDF 381件

主な公式出所：

- [1・2次の交付決定企業](https://seichotoushi-hojo.jp/1_2ji/koufu/)
- [3・4次の交付決定企業](https://seichotoushi-hojo.jp/koufu/)

データは必ず元の公式資料と検証状態を併せてご利用ください。

権利関係と免責については [`NOTICE.md`](NOTICE.md) を参照してください。
