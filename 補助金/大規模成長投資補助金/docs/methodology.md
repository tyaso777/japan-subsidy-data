# 作成方法

1. 公式公表ページから1次～4次のPDF URLを収集。
2. PDFをページごとに解析し、テキスト、表セル、図形・矩形座標を取得。
3. 長期成長ビジョン、外発的動機、内発的動機、売上成長目標、その他Boxを座標で分離。
4. 事業費・補助額は「事業費（補助額）」Boxを優先し、金額単位を百万円に正規化。
5. 事業費・補助額の案件代表値は、同一申請に含まれる複数投資案件の合計額とする。
6. 売上高は売上成長目標Boxを優先し、単体・連結、会社全体・補助事業等を別系列として抽出。
7. 申請企業代表は当社・単体・会社全体を優先し、PDF主系列と別IDで保持する。
8. 労働生産性、従業員給与、役員給与、従業員数は表セルを再構成し、年平均率と数式の整合性を検算。
9. 会社名ヘッダー混入やBox外文章の混入を除き、検証状態を各レコードに保持。
10. 全381案件・887ページを生成AIで画像目視し、売上主体、表の行列、役員給与の率のみ記載、投資内訳、期間原文を案件単位で再確認。
11. 横断QAで重複・欠落、売上算術、成長率定義、代表主体、2桁年度の過剰正規化を検査し、優先度の高い23案件を二巡目確認。

期間は原文ラベルを保持し、補正前・補正後を分けている。`23/12期`、`FY28`、`30年`等は分析用に2023、2028、2030へ補正し、`5年後`等は同じ系列の基準年がある場合に加算する。補正方法と信頼度も保存する。
