## 検証ルール

算術は、値がそろう場合だけ次式で確認した。

- 増加額: `target - baseline`
- 累積増加率: `(target / baseline - 1) × 100`
- 目標／基準比: `(target / baseline) × 100`
- CAGR: `(target / baseline) ^ (1 / 年数) - 1`
- 増加額の許容差: `max(0.05億円, 記載増加額の1%)`
- 成長率の許容差: 1.0ポイント

期間が2桁年・FY・相対表記の場合はCAGRを自動再計算せず、原文ラベルを残してnormalized yearをnullにした。
