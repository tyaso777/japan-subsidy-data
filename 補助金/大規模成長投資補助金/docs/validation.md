# 検証方法と注意点

## 実施済み

- 案件数、PDFページ数、Box数、指標行数の一致確認。
- 事業費・補助額のBox再抽出、補助率検算、補足金額を含むBoxの分類。
- 売上成長目標のBox選択、基準値・目標値からの成長率/CAGR検算。
- 複数売上系列の分離、原文期間の保持、基準＋増加額＝目標の算術検証。
- 申請企業代表は当社・単体・単独・会社全体・全社を優先し、補助事業、既存事業、親子・関連会社を除外。
- 通常企業の連結系列を申請企業代表から除外し、申請者がグループ・持株会社の場合のみグループ系列を候補化。
- PDF主系列と申請企業代表を別IDで保持し、一方で他方を上書きしない。
- 主要4指標の表セル再構成、基準年・目標年の順序、年平均率と計算値の差分確認。
- 役員給与の「絶対値なし・年平均率のみ」を別状態として保持。
- ページ内Box座標による文章分離と会社名ヘッダー除外。
- 全381案件・887ページを元PDF画像で目視確認。
- 全件監査後、売上・主要指標・年度表記の横断QAを実施し、優先度の高い23案件を独立して二巡目確認。
- `23/12期`、`FY28`、`30年`等は補正前をnullのまま残し、分析用補正後年を2023、2028、2030として保持。相対年は基準年がある場合だけ算出。
- 役員給与`rate_only`216件について、全件で年平均率が格納されていることを確認。

## 利用上の注意

検証状態が「記載なし又は抽出不能」「原本内不整合」等の場合は、欠損を0と扱わず、元PDFを参照してください。採否を説明する公式な審査データではないため、このデータ単独から「採択される数値」を断定できません。

生成AIによる画像読解は全案件の目視監査に使用しています。ただし、画像読解結果も絶対的な正解とはみなさず、PDF原文、`raw_evidence`、`corrections`、`confidence`を併記しています。分析時の具体的な期間・主体・欠損値の扱いは `analysis_data_handling.md` を参照してください。
