## 判定方法

1. `metrics.csv` の基準値・目標値・年平均率を、対応する目標値Boxのテキスト層に存在する数値と照合。
2. 不一致候補についてローカルPDFの該当ページを画像化。
3. 会社名、項目ラベル、単位、基準列・目標列の位置関係を含めて生成AIで目視確認。
4. 複数系列・括弧内内訳・ダッシュ・0を区別し、確定した誤りだけを修正。

注意: テキスト層に数値が見つからないだけでは誤りと判定していない。PDFの読み順や文字分割による偽陽性が多いため、画像Boxを最終根拠とした。
