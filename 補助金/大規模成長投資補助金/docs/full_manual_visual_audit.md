# 全381案件・全ページ画像目視監査

## 目的

機械抽出やテキスト順序だけでは、Boxの混線、OCRの桁落ち、複数企業・複数系列の取り違え、表見出しとセル内単位の不一致を十分に防げません。この再監査では、1次～4次の381案件について公開PDFの全ページを画像化し、1案件ずつ原寸相当で確認しました。

## 確認内容

- 会社名、ページ数、長期成長ビジョン頁、補助事業概要頁、追加頁
- 事業費、補助額、複数投資案件の合計と内訳
- 売上の主体、基準期、目標期、基準額、目標額、増加額、成長率、CAGR、倍率
- 労働生産性、従業員1人当たり給与、役員1人当たり給与、補助事業従業員数
- 表の原単位と万円換算値、年平均率だけが記載された役員給与
- 申請企業、グループ、親会社、子会社、関連会社、コンソーシアムの系列分離
- `yy/mm`、`yyyy/mm`、相対期間の原文と補正後年

## 出力

- `full_manual_visual_audit.jsonl`: 381案件の目視確認内容と修正根拠。381行、案件ID重複なし。
- `full_manual_visual_audit_changes.csv`: `cases.csv`、`sales_series.csv`、`metrics.csv`について、監査反映前後で実際に変わったセル。
- `cases.csv`: 申請企業を優先した横持ち代表値。申請企業の売上系列がない場合は、グループ値を無理に代表値へ入れず、`sales_reported_*`と`review_required`で扱う。
- `sales_series.csv`: 申請企業、連結、親子会社、補助事業等を系列別に保持。倍率は`growth_rate_pct`と分けて`sales_multiple`へ保持。
- `sales_series_annual.csv`: PDFに明記された中間年を含む年次点。

## 年度の扱い

`24/3期`は原文ラベルをそのまま残し、`year_before_correction=24`、`year_after_correction=2024`、月は`3`として保持します。`30/7期`も同様に2030年7月期へ補正します。`5年後`や`事業化報告3年目`は、同じ系列の基準年が確定できる場合だけ西暦へ変換し、確定できない場合は相対ラベルを残します。

## 分析上の注意

- 逆算した売上額はPDFに直接印字された値ではありません。増加額と率の丸めによって、表示値どうしに小さな差が生じます。
- `成長率`が「増加率」か「目標÷基準の到達指数」か、PDF内で曖昧な例があります。`sales_growth_rate_definition`を必ず併用してください。
- 「倍」はパーセントと別です。1.7倍は到達指数170%、累積増加率70%です。
- 表の行見出しが万円でも、セル自体に千円と明記された例があります。原単位列を優先し、換算列と併用してください。
- 複数企業の表では`cases.csv`だけで全系列を再現できません。詳細比較には`sales_series.csv`と監査台帳を併用してください。
- 元PDF自体に事業費の見出し値と明細合計、率と絶対額などの不整合がある場合は、推測で一方へ統一せず監査注記に残しています。

## 再適用

```powershell
python scripts/apply_full_manual_audit.py --project-root . --ledger-dir <監査JSONLのフォルダ>
```

この処理はPythonだけで、CSV、互換CSV、`html/data/cases.json`、`qa.html`、`index.html`を同期します。

## 全ページ突合の最終結果

- 381 PDF、合計887ページについて、PDF実ページ数と監査台帳の`pages_reviewed`／`pages_checked`を機械的に突合し、不一致0件を確認しました。
- 売上は508系列、主要指標は1,556レコード、年度別売上は466点です。
- 追加ページの記録が不足していた9案件（西部技研、奥伊吹観光、古川電気工業、上野ロジケム、セントラルフード、ファーストライン、MHT、明和工業、メタルテック）は追加ページを再度画像目視し、監査台帳へ確認ページと所見を追記しました。
- 古川電気工業は、コンソーシアムと古川工業の売上増加額を基準売上として扱っていた誤りを修正し、古川工業の費用・主要4指標を別系列で追加しました。
- 明和工業は、ヨシカワ機械の主要4指標を別系列で追加し、2030年目標とBox見出し2031年3月期の不一致を併記しました。
- 奥伊吹観光は、冬期・夏期事業の2029年売上点を追加しました。
- MHTは、追加ページにある設備別14項目（合計1,500百万円）を保持し、事業費総額1,524百万円との差24百万円を原資料内の未明示差額として注記しました。
- アイサービスの原文233%は累積増加率ではなく目標／基準比（約2.33倍）として扱い、分析用累積増加率133%と分離しました。

PDF内で丸め値や概数が併記されるため、基準額・目標額・増加額が完全一致しないケースがあります。`arithmetic_status`が「PDF原文内不整合」「丸め差」等の場合は抽出誤りではなく、原文値をすべて保持したうえで不整合を明示しています。
