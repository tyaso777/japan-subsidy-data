# 公式公募回別代表値の出典

`data/reference/official_round_benchmarks.csv` は、公式が公表した申請者全体・採択者の代表値を公募回別に転記した参照表です。原則は中央値ですが、公式注記に従い「全社売上高に対する補助事業売上高の割合」だけは平均値として `statistic=mean` を設定しています。

## 使用した公式URL

| 対象 | 公式資料 | 主な用途 |
|---|---|---|
| 1次 | https://seichotoushi-hojo.jp/assets/pdf/ichiji.pdf | 1次個別表との照合 |
| 2次 | https://seichotoushi-hojo.jp/assets/pdf/niji.pdf | 2次個別表との照合 |
| 1・2次合算 | https://seichotoushi-hojo.jp/assets/pdf/information/20250122.pdf | 1・2次合算値との照合 |
| 3次 | https://seichotoushi-hojo.jp/assets/pdf/3ji_median.pdf | 3次個別表、および統合表にない全社賃上げ予定率 |
| 4次 | https://seichotoushi-hojo.jp/assets/pdf/4ji_median.pdf | 4次個別表、および統合表にない全社賃上げ予定率 |
| 1～4次統合 | https://chukentou-seichotoushi-hojo.jp/assets/lp/documents/000058716.pdf | 1～4次の最新統合値を優先採用 |
| 5次 | https://chukentou-seichotoushi-hojo.jp/assets/lp/documents/5ji_median.pdf | 5次の全指標 |

## 出典の採用方針

1. 1～4次については、2026年3月9日開催の公式説明資料 `000058716.pdf` に掲載された統合表を原則として優先します。
2. 統合表に掲載されていないが個別公式資料に掲載されている指標は、個別資料を補完出典として使用します。
3. `source_url` は各CSV行の実際の転記元を示します。補完値には統合表ではなく個別資料のURLを記録します。
4. 過去資料と最新資料で値が異なる場合は、最新統合表の値を採用します。例として、3次の「従業員給与支給総額の増加額（申請者全体）」は個別資料の2.5億円ではなく、最新統合表の2.4億円を採用しています。

## 全社賃上げ予定率の補完

2026年3月の1～4次統合表では「全社賃上げ予定率」が省略されています。一方、3次・4次の個別公式資料には掲載されているため、次の2行を個別資料から補完しています。

| 公募 | 申請者全体中央値 | 採択者中央値 | 単位 | 出典 |
|---|---:|---:|---|---|
| 3次 | 2.4 | 2.4 | % | https://seichotoushi-hojo.jp/assets/pdf/3ji_median.pdf |
| 4次 | 2.5 | 2.4 | % | https://seichotoushi-hojo.jp/assets/pdf/4ji_median.pdf |

この指標は、公式注記上「最新決算期と基準年（補助事業完了日を含む事業年度）の数値を比較した率」です。現在の案件データには直接対応する全社賃上げ予定率の列がないため、`current_field` は空欄、`comparability` は `unavailable` としています。

## 7指標プロットでの企業値

`html/public_metrics_dashboard.html` では、公開PDFから作成できる7指標を横軸・縦軸に選択できます。うち次の2指標は公式申請値そのものではなく、公開値からの近似です。

- 従業員給与支給総額の増加額（億円）＝｛（目標1人当たり給与（万円/人）×目標従業員数）－（基準1人当たり給与（万円/人）×基準従業員数）｝÷10,000
- 全社売上高に対する投資額割合（%）＝公開された補助事業の事業費（百万円）÷公開された基準売上高（億円）

前者は公開PDFの従業員数が全従業員数と一致しない場合があるほか、元の数値が丸められています。後者は公式指標が「最新決算期の全社売上高」を分母とする一方、公開値では基準売上高を使用するため、いずれも `comparability=proxy` としています。

## 比較上の注意

公式表の「採択者」と、本データセットが収録する「公式サイトで取組概要PDFが公開された交付決定企業」は同一母集団とは限りません。また、公開PDFから再構成した企業別値は、丸め、対象範囲、代表企業・拠点の選択によって公式申請値と差が生じる可能性があります。
