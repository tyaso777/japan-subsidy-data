# 成長投資計画シミュレーター

大規模成長投資補助金6次公募の様式に沿って、過去実績と目標値から将来PLを検討するシミュレーターです。

## 使い方

`成長投資計画シミュレーター_大規模成長投資補助金6次公募.html` をブラウザで直接開けば、ローカルサーバーなしで使用できます。

## 指標定義

- 数式・ロジック画面に第3次定義と第6次定義を併記
- 計画値、達成判定、目標への自動調整は第6次事前公開版の定義を使用
- 前々期・前期・最新決算期の過去3期を入力
- 最新決算期の翌期から、補助事業完了年度（基準年）と事業化報告3年目までを管理（将来最大9期）
- 第6次様式に合わせ、FTE従業員数と役員数を分けて管理
- 過去中央値は旧定義を含むため、目標水準の参考値として使用

## 開発・検証

```bash
npm install
npm test
npm run build:standalone
```

`npm run build:standalone` で単独HTMLを再生成します。
