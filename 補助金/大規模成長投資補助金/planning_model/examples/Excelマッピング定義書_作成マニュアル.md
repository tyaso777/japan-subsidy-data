# Excelマッピング定義書 作成マニュアル

## 目的

このシミュレーターと任意形式のExcel（`.xlsx` / `.xlsm`）のセルを、JSONのマッピング定義書で結びます。出力時は元ファイルを上書きせず、書式・数式・マクロ・非対象セルを保持した別Excelを保存します。

## Copilotへの依頼例

対象Excel、このマニュアル、同じフォルダのJSONサンプルをCopilotへ渡し、次のように依頼します。

```text
添付したExcelを確認し、添付した「Excelマッピング定義書 作成マニュアル」とJSONサンプルに従って、成長投資計画シミュレーター用のマッピング定義書を作成してください。

要件:
1. 出力は growth-investment-excel-mapping/v1 形式のJSONファイルとしてください。
2. 対象Excelのシート名とセル番地を実際に確認し、計算式セルではなく入力セルを指定してください。
3. 各セルの金額単位と、パーセントがExcel内部で0.05と保持されるか5と保持されるかを確認してください。
4. 空欄と数値の0を区別してください。
5. 読み込みだけなら direction を import、書き出しだけなら export、両方なら both としてください。
6. 対応関係を推測した項目、候補が複数ある項目、対応できない項目は、JSONとは別に一覧で報告してください。
7. 元のExcelは変更しないでください。

最初に、対象Excelで確認できたシート名と候補セルの一覧を示してください。その後に完成したJSONを提示してください。
```

生成されたJSONは、人がシート名・セル番地・単位を確認してから使用します。

## target

安定識別子は `データ区分.期.第6次様式番号` です。

| 要素 | 指定値 |
|---|---|
| データ区分 | `balanceSheet` / `companyPL` / `projectPL` |
| 期 | `prePrevious`（前々期）/ `previous`（前期）/ `latest`（最新決算期） |
| 第6次様式番号 | `1-1`、`2-1`、`7-1` など |

例：`companyPL.latest.2-1` は「最新決算期・全社PL・2-1 売上高」です。

現時点の対象は、過去3期のB/S、全社PL、補助事業PLの手入力項目です。自動計算項目は入力先にできません。

## excel

| キー | 内容 |
|---|---|
| `sheet` | Excelの正確なシート名 |
| `cell` | A1形式のセル番地 |
| `unit` | `raw` / `円` / `千円` / `百万円` / `億円` / `%` / `人` / `年` / `倍` |
| `percentMode` | 5%をExcel内部で0.05として持つ場合は `fraction`、5として持つ場合は `display` |

## direction

- `import`: Excelからシミュレーターへの取込のみ
- `export`: シミュレーターから別Excelへの出力のみ
- `both`: 双方向（省略時）

## transform

単位変換後に `値 × scale + offset` を適用します。符号を逆転する場合は `invertSign: true`、丸めは `round` で指定します。

## 安全上の制約

- JSON以外のスクリプトや数式は実行しません。
- 空欄と0は区別します。空欄は変更せず、0は明示的な0として反映します。
- 出力先に数式がある場合は停止します。
- 出力先セルはテンプレート内にあらかじめ存在する必要があります。
- 同じ出力セルを複数定義できません。
- `.xls`（旧バイナリ形式）は対象外です。`.xlsx` または `.xlsm` に変換してください。

## 運用手順

1. マッピング定義書と対象Excelを選択します。
2. 取込時は「取込内容を確認」で差分とエラーを確認します。
3. エラーが0件の場合だけ「確認した値を反映」を押します。
4. 出力時は「別Excelとして出力」を押します。元Excelは変更されません。
