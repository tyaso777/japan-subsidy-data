# 定量検証メモ（申請公募回補正後28件）

このメモは、`applicant_median_residual_report.html` の定量分析を再現するための技術付録である。結論と解釈はHTMLレポートを正本とし、本書は定義、感度分析、出力ファイルを簡潔に記録する。

## 1. 分析対象

母集団は、公開企業PDFを収録した採択企業381レコードである。現行9指標（No.1、2、7、8、9、10、11、13、14）のうち、次の条件を満たす企業を対象とした。

- 同じ申請公募回の「申請者全体中央値」と企業値を比較する。
- 比較可能な指標が5つ以上ある。
- 比較可能な指標の60%以上で、企業値が申請者全体中央値を下回る。
- 同値は「下回る」に含めない。

判定可能373件中28件（7.5%）が該当する。主要品質フラグのない範囲では199件中16件である。

## 2. 申請公募回の補正

元データの1次・2次 `round` は、申請公募回ではなく企業PDFの公開・命名バッチを反映していた。公式採択一覧の企業名を照合し、旧社名2件を法人番号で確認した結果、対象181件中60件の回次を補正した。

- 補正前の対象：29件
- 補正後の対象：28件
- 除外：アサヒセイレン中部、八立製作所
- 追加：浦島観光ホテル
- 補正後の回次構成：1次3件、2次2件、3次8件、4次15件

固定クロスウォークは `application_round_crosswalk_1_2.csv`、対象集合への影響は `round_correction_impact.csv` に保存した。元の `cases.csv` は変更していない。

## 3. 境界と指標集合への感度

「中央値未満」の境界を変えると、該当数は次のように変わる。

| 未満とみなす境界 | 該当数 |
|---|---:|
| 0%（企業値 < 中央値） | 28 |
| 1%以上の下振れ | 27 |
| 5%以上の下振れ | 20 |
| 10%以上の下振れ | 14 |

28件の観測208セル中25セルは、申請者中央値との差が5%未満である。したがって、28件すべてを同じ強度の「定量劣後」とは扱わない。

指標集合を変更すると対象企業も動く。現行9指標の28件に対し、No.13除外では30件（共通28件）、No.8除外では28件（共通19件）、No.14除外では26件（共通18件）、従来7指標では30件（共通15件）である。現象の件数は比較的残る一方、企業別メンバーシップはNo.8・No.14の有無に敏感である。

## 4. 偶然期待との比較

各公募回・各指標で実際に観測された中央値未満率を固定し、企業への未満フラグを指標ごとに無作為再配置する置換検定を20,000回行った。

- 全件：観測28件、偶然期待平均14.30件、95%範囲8～21件、上側p=0.00030
- 主要品質フラグなし：観測16件、偶然期待平均8.92件、上側p=0.0051

これは「中央値未満の指標が同一企業に集積している」ことを示す。ただし置換は指標間の実際の相関を壊すため、採択因果を立証する検定ではない。

## 5. 28件に残る定量的な強み

28件すべてに、申請者全体中央値以上の指標が少なくとも1つ残る。最も厳しい3件は、いずれも6指標を観測でき、そのうち5指標が中央値未満、中央値以上は1指標だけである。

- シュゼット：No.13のみ中央値以上。No.13は方向解釈に注意が必要。
- 北海道ちぬやファーム：No.11のみ中央値以上。
- レクザム：No.2のみ中央値以上。

したがって「全指標で申請者中央値を下回る採択企業」ではなく、「公開9指標の多くで下回るが、一部の強みを残す採択企業」が正確な記述である。

## 6. 規模・費用対効果・構造仮説

同じ公募回・業種・規模に近い採択企業を1対1で非復元マッチングした。

- 事業費と補助金額には明確な差がない。
- 従業員増加数と補助金額当たり従業員増加数は28件側で小さい。
- 補助金額当たり給与総額増加も28件側で小さい。
- 共同申請は28件中7件（25.0%）、同回その他では10/345件（2.9%）。同回置換p=0.00025、探索39特徴のBH補正q=0.00975。
- 本社所在地が関東の案件は28件で0件、その他で27.8%。同回置換p=0.0019、BH補正q=0.0370。

共同申請は「単独企業の公開指標では表れない参加者間・供給網効果」の候補だが、共同申請自体が採択理由だとは断定できない。地域差も公募回・産業構成との交絡を完全には除いていない。

基準従業員数は28件で小さいが、No.8・No.10の推計式にも従業員数を用いるため、対象抽出との循環がある。規模差を独立した採択要因として扱わない。

## 7. 限界

- 不採択企業の企業別指標、申請書全文、審査点、口頭審査、金融機関資料は非公開である。
- No.8、No.10、No.13、No.14には公開PDFからの近似が含まれる。
- 「申請者全体中央値」は群の基準値であり、採択閾値ではない。
- 28件は採択企業内で定義した事後的な部分群であり、探索結果を採択確率に読み替えない。
- マッチング対照も採択企業であるため、採択／不採択の因果比較ではない。

## 8. 主な出力

- `core28_company_quantitative_table.csv`：企業別値、同回申請者中央値、比率、規模、対照企業
- `application_round_crosswalk_1_2.csv`：1・2次の申請公募回クロスウォーク
- `round_correction_impact.csv`：補正前後の対象集合差分
- `boundary_sensitivity.csv` / `boundary_metric_cells.csv`：境界感度
- `metric_set_sensitivity.csv` / `metric_set_membership.csv`：指標集合感度
- `permutation_test_summary.csv` / `permutation_test_distribution.csv`：置換検定
- `matched_accepted_pairs.csv`：同回・近似規模の採択企業対照
- `matched_scale_and_effect_comparison.csv`：マッチ後の規模・効果比較
- `round_adjusted_structure_geography_comparison.csv`：構造・地域探索
- `hypothesis_evidence.csv` / `summary.json`：仮説別の機械可読要約
- `quantitative_source_manifest.csv`：定量出力の役割とSHA-256
