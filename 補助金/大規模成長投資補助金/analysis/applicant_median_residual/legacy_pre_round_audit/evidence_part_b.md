# 申請者全体の公式代表値を複数指標で下回る交付決定企業：定性精査 Part B

> **履歴資料（非正本）。** 本ファイルは申請公募回の監査前に抽出した29件のうち後半14件を分担精査した記録である。最終対象は親フォルダの `qualitative_coding_28.csv` と `applicant_median_residual_report.html` を参照すること。アサヒセイレン中部・八立製作所は最終対象から除外し、浦島観光ホテルは親フォルダの `evidence_urashima.md` で別途精査した。個別PDFから読み取った事実と採点は履歴保持のため残している。

## 1. 対象と分割方法

`case_level_reassessment.csv` から、次の条件をすべて満たす29企業レコードを抽出した。

- 現行9指標のうち5指標以上を観測できる。
- 同じ公募回の申請者全体の公式代表値を厳密に下回る指標が、観測指標の60%以上である。

29件を `case_id` の文字列昇順で安定ソートし、本ファイルは順位16～29の14件を扱う。対象 `case_id` は次のとおり。

```text
s2_outline_19, s2_outline_192, s2_outline_194, s2_outline_195,
s2_outline_205, s2_outline_210, s2_outline_218, s2_outline_221,
s2_outline_232, s2_outline_24, s2_outline_35, s2_outline_72,
s2_outline_95, s2_outline_99
```

企業属性・事業費・補助額・URLは `data/processed/cases.csv`、ページ本文と役割は `data/text/pages.jsonl`、原資料の視覚確認は `local_assets/pdfs/<case_id>.pdf` を使用した。

## 2. コーディング規則

### 2.1 A～Hの点数

| 点 | 判定 |
|---:|---|
| 0 | 公開範囲に記載がない、または当該要素に該当しない |
| 1 | 一般論・方針・市場トレンドはあるが、案件固有の裏付けが弱い |
| 2 | 案件固有の事実や数値があるが、契約・能力差・許認可・資金等の決定的証拠が不足する |
| 3 | 実名主体、実績数量、物理的能力上限、取得済み用地、契約、設備仕様等、第三者が照合しやすい具体証拠がある |

要素は次の9種類である。

- A：実名・数量付き需要確度
- B：物理的能力制約
- C：事業転換の大きさ
- D：供給網・地域の不可欠性
- E：実行準備（用地・許認可・見積・導入実績等）
- F：財務・自己負担能力
- G：雇用・賃上げの質
- H：政策適合・波及性
- I：A～Hをこの29件に特有の採択説明とみなしにくくする反証

### 2.2 Iの点数

Iだけは高得点ほど慎重な解釈を要する。

| 点 | 判定 |
|---:|---|
| 0 | 公開資料内に明確な反証が見当たらない |
| 1 | 一部の根拠が一般的、または顧客・資金等の一部が非公開 |
| 2 | 主要根拠が市場一般で、契約・数量・資金等の重要情報が不足する |
| 3 | 未確定事項が明記される、または案件固有の需要・能力・資金根拠が薄く、一般的な採択案件記述との差別化が難しい |

### 2.3 推論確度

`High / Medium / Low` は、「公開PDFの事実が、公開定量値以外の評価候補をどの程度明瞭に支えるか」の確度である。実際の採択理由や因果効果の確度ではない。非採択企業の個票、審査得点、面接内容は公開されていないため、「この理由で採択された」とは判定しない。

## 5. Part Bだけから得られる暫定的観察

この14件では、定量指標の低さを一つの共通要因だけで説明できない。比較的具体性が高いのは、次の三類型である。

1. **実測可能な能力制約**：15から22バッチ/日、火入れ400から1200本/時、精米能力2.6倍、冷凍能力1.5倍、借地更新不能・排水能力上限など。
2. **事業モデルの転換**：卸売から製造、運送から3PL、小売から食品製造、外部委託から自社一貫、百貨店からホテルなど。
3. **地域インフラとしての案件固有性**：食料・農業供給、中心市街地再生、文化財・観光、地域サプライヤー不足など。

一方で、財務・自己負担能力の公開根拠は全体に弱く、顧客別契約・受注数量がない案件も多い。したがって、A～Hの高得点を「この要素が採択をもたらした」とは読まない。公開資料で説明できない部分は、申請書本体、金融機関確認、審査得点、プレゼンテーションに残る可能性がある。
