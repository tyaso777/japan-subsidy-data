# 申請者中央値を複数指標で下回る採択企業：定性精査 Part A

> **履歴資料（非正本）。** 本ファイルは申請公募回の監査前に抽出した29件のうち前半15件を分担精査した記録である。最終対象は親フォルダの `qualitative_coding_28.csv` と `applicant_median_residual_report.html` を参照すること。アサヒセイレン中部・八立製作所は最終対象から除外し、浦島観光ホテルは親フォルダの `evidence_urashima.md` で別途精査した。個別PDFから読み取った事実と採点は履歴保持のため残している。

## 1. 対象と読み方

対象は、現行9指標のうち5指標以上を観測でき、同じ公募回の申請者全体中央値を下回る指標が60%以上ある29企業レコードである。29件を `case_id` の昇順で安定ソートし、前半15件を本ファイルで扱う。

- 対象順位：1～15
- 根拠資料：`data/text/pages.jsonl` と `local_assets/pdfs/` の企業別公開PDF
- 分析単位：企業レコード。共同申請では申請案件数と一致しない
- `is_clean=False` は、今回の4件ではいずれも `RATE_DEFINITION_AMBIGUOUS`。定性記載が誤りという意味ではなく、率指標の比較に品質上の注意がある

公開PDFは採択後の要約資料であり、審査時の申請書全文、審査得点、非採択企業の資料ではない。したがって、以下の「評価候補」は採択理由の確定ではない。事実と推論を分け、推論確度を付した。

## 2. コーディング基準

### A～H（正方向の証拠）

| 点 | 意味 |
|---:|---|
| 0 | 公開資料に具体的な記載がない |
| 1 | 一般論・意向・市場トレンドの記載にとどまる |
| 2 | 数量、工程、主体、制度、実績のいずれかが具体的 |
| 3 | 実名顧客・受注、需給差、能力上限、取得済み用地、見積、既存導入実績など、検証可能性が特に高い |

項目は、A＝実名・数量付き需要確度、B＝物理的能力制約、C＝事業転換の大きさ、D＝供給網・地域の不可欠性、E＝実行準備、F＝財務・自己負担能力、G＝雇用・賃上げの質、H＝政策適合・波及性である。

### I（反証・一般性の注意）

Iだけは高得点ほど注意が強い。

| 点 | 意味 |
|---:|---|
| 0 | 明確な反証なし。ただし非採択比較がない限界は残る |
| 1 | 個別性の高い証拠があるが、採択企業のみの資料である |
| 2 | 具体性は一部あるが、他の採択案件にも頻出する説明が中心 |
| 3 | 顧客・数量根拠が弱い、テンプレート残りがあるなど、個別説明として明確な弱点がある |

`dominant_archetype` と旧1～4次の審査項目との対応は分析者の推論であり、実際の審査得点を表さない。

## 4. 企業別根拠

### 1. 株式会社松尾製作所（`s1_outline_50`）

**事実。** 顧客から引き合いのある次世代電動車部品59品番について専用工程を計画している。組付けは従来品の2倍以上の部品、約3倍のステップを要し、AI検査、レーザー溶接、ネック工程の冗長化を組み込む。[p.2 / 補助事業の概要]

**推論。** 顧客名や受注量は見えないが、品番と工程の対応が具体的であるため、「需要に対して何を作るか」という技術的実装性が評価候補となる。電動化、自動化、人手不足は他の採択案件にも多く、これだけで採択を説明したとは言えない。確度 **Medium**。

### 2. 株式会社オティックスホールディングス（`s1_outline_63`）

**事実。** 電動車部品売上を82億円から2030年181億円、2038年600億円へ伸ばし、エンジン部品メーカーからモビリティ部品メーカーへ転換する。第1工場は自己資金で完成済みで、第2工場は「受注した電動車部品」の量産を担う。[p.1 / 長期成長ビジョン、p.2 / 補助事業の概要]

**推論。** 受注済み需要、先行自己投資、主力事業転換が同時に確認でき、公開定量指標以外の評価候補として比較的強い。確度 **High**。

### 3. 千寿製薬株式会社（`s1_outline_77`）

**事実。** 現有能力7,000万本に対し、新薬を含む需要を2027年度7,500万本、2031年度1億本超と見込む。3,500万本の新ラインを追加し、既存薬の安定供給と新薬生産、保守時の冗長性を確保する。[p.2 / 補助事業の概要]

**推論。** 需給差を同一単位で示した能力制約は明瞭である。ただし需要は社内販売予測で、顧客契約や公的需要予測による外部検証はできない。確度 **High**。

### 4. アサヒセイレン中部株式会社（`s1_outline__179`）

**事実。** 国内アルミ需要を2019年4,218千tから2050年6,020千t、輸送用を1,700千tから2,900千tと見込み、環境配慮型合金、スクラップ選別、ドロス処理を需要地近くで増強する。国内で稀有なアルミ総合リサイクルメーカーと自社を位置づける。[p.1 / 長期成長ビジョン、p.2 / 補助事業の概要]

**推論。** 国内循環供給のハブ機能は評価候補だが、顧客名、注文量、現有能力差はない。率定義にも品質注意がある。確度 **Medium**。

### 5. 株式会社ホテルニューアワジ（`s1_outline__181`）

**事実。** 島内の外国人宿泊比率1.0％という課題を起点に、2社・2市で外国人向けホテルを建設する。客室単価、室数、稼働率90％から売上を算定するほか、自治体、農漁業者、観光協会等との事業、認証、既存セントラルキッチン・省人化運営実績を示す。[p.1、3-5 / 長期成長ビジョン、p.6、9 / 補助事業の概要]

**推論。** 単体ホテルではなく、食、移動、滞在、雇用を面的に結ぶ地域プラットフォーム性と既存実行基盤が評価候補である。外国人需要は将来予測であり、予約実績ではない。確度 **High**。

### 6. 株式会社八立製作所（`s1_outline__196`）

**事実。** コマツ大阪工場のブラケット部門で約8割のシェアを持つ。工程分散と低生産性を、新工場、粉体塗装、レーザー加工等で解消し、国内供給網を維持する。[p.1 / 長期成長ビジョン、p.2 / 補助事業の概要]

**推論。** 本件は観測8指標すべてが申請者中央値未満であり、公開定量だけでは最も説明しにくい。約8割シェアという既存供給ポジションと国内代替困難性が有力候補だが、発注量・審査点は不明である。確度 **High**。

### 7. 株式会社コスモス食品（`s2_outline_115`）

**事実。** 既存設備が製造量の限界にあり、真空凍結乾燥機で能力を40％増やす。倉庫、搬送、検査、生産管理を更新し、賃金年平均9.1％を計画する。[p.2 / 補助事業の概要]

**反証。** 顧客、受注、市場数量の根拠がなく、公開スライドに `XX`、修正メモ等が残る。[p.1-2 / 公開概要全体]

**推論。** 能力制約と設備機序は具体的だが、本件特有の採択説明としては弱い。確度 **Medium**。

### 8. 奥伊吹観光株式会社（`s2_outline_116`）

**事実。** 年間25万人が来場するが9割が日帰りで、宿泊施設を持たず、客単価は約7,500円にとどまる。50室・118名のホテル、冬夏の稼働率、イベント来場実績を基に通年運営を計画する。[p.1 / 長期成長ビジョン、p.2-3 / 補助事業の概要]

**推論。** 既存集客を宿泊消費へ転換する収益モデルと、宿泊能力ゼロという制約が明確である。予約・契約による需要保証ではない。確度 **High**。

### 9. 株式会社ヤマキ食品（`s2_outline_121`）

**事実。** 同業撤退による需要集中、既存工場の能力不足を、新工場・自動化・多品種少量・PBで解く。契約農家への収穫機貸与、若手育成、ISO22000、3社見積比較を示す。[p.1 / 長期成長ビジョン、p.2 / 補助事業の概要]

**推論。** 顧客需要の数量よりも、見積済みの実行準備と地域農業を含む供給体制の一貫性が評価候補である。確度 **High**。

### 10. 株式会社 博多屋（`s2_outline_137`）

**事実。** 自社既存ホテルの隣接所有地に98室の新館を建てる。敷地・延床面積、2026年着工・2027年竣工、ADR9,500円、既存ホテル改修実績、地域観光団体での役職を示す。[p.1 / 長期成長ビジョン、p.2 / 補助事業の概要]

**推論。** 需要は地域の宿泊者回復・政策目標であり予約根拠ではない。一方、所有地と工期、既存運営実績は実現可能性の評価候補となる。率定義に品質注意がある。確度 **Medium**。

### 11. 株式会社サン海苔（`s2_outline_149`）

**事実。** 製造能力不足による機会損失、人手工程、HACCP未対応を新工場・自動化で解く。同社は佐賀海苔生産者の出資で設立され、生産者への利益還元と価格安定を掲げる。[p.1 / 長期成長ビジョン、p.2 / 補助事業の概要]

**推論。** 地域生産者との所有・利益関係は固有性があるが、需要・失注・能力不足の数量はない。確度 **Medium**。

### 12. 株式会社 ファーストライン（`s2_outline_159`）

**事実。** 新規顧客と契約協議中で、超低温・冷凍・冷蔵・定温・常温の5温度帯倉庫を建てる。和歌山県初・県内唯一とし、既存の医薬品・食品配送経験を示す。[p.1 / 長期成長ビジョン、p.2-4 / 補助事業の概要]

**推論。** 地域物流インフラとしての希少性は高いが、顧客名・契約量はなく、需要は協議段階である。率定義にも品質注意がある。確度 **Medium**。

### 13. 明和工業株式会社（`s2_outline_168`）

**事実。** 水道中心から産業設備ユニットの総合エンジニアリングへ転換する。子会社工場は面積・能力上限に達し、NC旋盤13機中7機が1992～97年製。隣接地を取得済みでボーリング調査中、新工場で床面積約3倍、能力2倍を計画する。[p.1 / 長期成長ビジョン、p.2-3 / 補助事業の概要]

**推論。** 事業転換、M&Aによる垂直統合、物理的上限、取得済み用地が一つの実行計画につながっている。顧客名・受注額は不明。確度 **High**。

### 14. オギハラ食品株式会社（`s2_outline_171`）

**事実。** 外食チェーンの新規出店に伴う増産要求、2工場分散、人手の異物除去、FSSC22000対応を挙げる。国内トップシェアとし、三池高菜生産者への利益還元、従業員58人から91人を計画する。[p.1 / 長期成長ビジョン、p.2 / 補助事業の概要]

**反証。** 顧客・増産量は非開示で、公開資料に「数値確定後記入」が残る。率定義にも品質注意がある。[p.2 / 補助事業の概要]

**推論。** 地域食文化・生産者との関係は評価候補だが、需要確度と資料完成度が弱く、確度 **Medium**。

### 15. 株式会社環境整備産業（`s2_outline_172`）

**事実。** TSMCが大分の処理施設を視察し、既存施設では処理依頼を断る事例が頻発する。県内廃棄物のうち感染性65.8％、廃酸・廃アルカリ61.3％が県外処理である。M&Aで特別管理廃棄物ノウハウを得ており、保有遊休地7,060㎡に県内最大規模の施設を建てる。[p.1 / 長期成長ビジョン、p.2 / 補助事業の概要]

**推論。** 実需、地域の処理不足、取得済みノウハウ・用地が同時に確認でき、公開定量外の評価候補として強い。ただし非採択案件との比較と審査点はない。確度 **High**。

## 5. Part Aだけから得られる暫定所見

1. 15件の主類型は、供給網・地域インフラ8件、事業転換・垂直統合3件、需要確約・能力制約3件、実行準備・財務1件であった。
2. 需要を顧客名・受注・需給差で強く示せる案件は一部に限られる。オティックス、千寿製薬、八立製作所、環境整備産業が相対的に強い。
3. 財務・自己負担能力の具体記載はほぼない。公開要約から財務審査を逆算することはできない。
4. 地域・供給網の説明は多いが、一般的な「地域貢献」と、シェア・県外流出率・所有関係で示す代替困難性を区別する必要がある。
5. コスモス食品とオギハラ食品にはテンプレート残りがあり、公開資料の文章完成度そのものが採択を説明しない反証となる。
6. 以上は「採択された企業に見られる説明候補」であり、「この特徴があるから採択される」という識別結果ではない。
