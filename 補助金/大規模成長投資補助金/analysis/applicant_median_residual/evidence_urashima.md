## 3. PDFに記載された事実

### 3.1 事業全体

1. 浦島観光ホテル、丸峰観光ホテル、みちのりホテルズの3社が、各地域を代表するフラッグシップホテルとして高付加価値化と生産性改善を行う構想を示す。コンソーシアム全体の売上は2023年度55.6億円から2029年度82.0億円、47%増を目標とする。[p.1 / 長期成長ビジョン]
2. ハード面では、老朽施設・設備と団体客向け設備が個人客・インバウンド需要に合わず、従業員工数も増えていると説明する。[p.1 / 長期成長ビジョン]
3. ソフト面では、人手不足が施設稼働率の引上げを制約し、従業員の3割弱から3割が60歳以上で、稼働調整、業務効率化、省人化が必要だとする。[p.1 / 長期成長ビジョン]

### 3.2 浦島観光ホテル

1. 4施設は開業から30～60年が経過し、客室、バックヤード、設備全体の老朽化で価格競争力が低下したと記載する。[p.2 / 補助事業の概要]
2. 団体旅行時代の客室・共用部を個人利用向けに転換し、内装、レストラン、温泉脱衣場、衛生インフラ、省エネルギー設備を更新する。[p.2 / 補助事業の概要]
3. PMS、自動チェックイン・アウト、カードキーを導入し、ホテル内の3フロントを1か所に集約して人員を館内案内へ再配置する。[p.2 / 補助事業の概要]
4. 事業費69億円、補助額12億円、労働生産性730万円/人から999万円/人、給与326万円/人から385万円/人、従業員236人据置きを掲げる。[p.2 / 補助事業の概要]

### 3.3 丸峰観光ホテルとみちのりホテルズ

1. 丸峰観光ホテルは2023年2月に民事再生を申し立て、JPiXグループが2024年1月に事業承継した。前オーナー時代の資金不足で未改修箇所が多数残ると記載する。[p.3 / 補助事業の概要]
2. 丸峰は芦ノ牧温泉地区で最大規模の客室数を持つとし、個人客、インバウンド、若年層、ペット愛好家、歴史愛好家を対象に改修する。[p.3 / 補助事業の概要]
3. みちのりホテルズは最後の大規模改修から約10年が経ち、老朽化に伴う保守工数と顧客満足維持のサービス工数が増えたとする。露天風呂付き客室、カードキー、PMS、清掃ロボット、配管改修を計画する。[p.4 / 補助事業の概要]
4. 公開PDFの記載額を合算すると、3社合計の事業費は75.9億円、補助額は14.3億円である。差額61.6億円は補助額以外で賄う必要があるが、その調達手段や確約状況は公開されない。[p.2-4 / 補助事業の概要]

## 5. 事実・推論・反証の分離

### 事実

- 築30～60年の4施設、3フロント分散、個別設備、企業別の事業費・補助額・賃金・人数は公開PDFに明記される。
- 丸峰観光ホテルの民事再生とJPiXグループによる事業承継は公開PDFに明記される。
- 記載額の単純合算では、3社合計事業費75.9億円、補助額14.3億円、差額61.6億円となる。

### 推論

- 3地域の既存ホテル群を、団体旅行型の老朽施設から個人・インバウンド向け高付加価値型へ一体転換することは、公開定量指標以外の評価候補になり得る。
- 宿泊需要の新設ではなく、既存客室・温泉・立地・運営組織を再構成する案件であるため、全くの新規参入より実行可能性が評価された可能性がある。
- 人手不足が稼働率を制約するという因果鎖に対し、フロント集約、セルフ機器、PMS、清掃ロボットを対応させており、設備と課題の対応関係は評価候補になり得る。
- ただし、以上は採択後の公開要約から読み取れる候補であり、実際の審査理由ではない。

### 反証・未確認事項

- 顧客名、予約、稼働率、ADR、需要量、第三者市場調査は示されない。
- 補助外負担61.6億円の資金源、融資確約、自己資金余力は示されない。
- 地元調達、自治体・DMO・交通事業者との協定、域内消費や雇用純増の波及量は示されない。
- 丸峰は民事再生後であり、承継スポンサーの存在は実行基盤の候補である一方、財務リスクが解消した証拠にはならない。
- 比較対象は採択・交付企業の公開資料だけで、非採択企業の申請書、審査点、面談記録はない。

## 6. 類型と評価候補

- 主類型：**事業転換・垂直統合**
- 旧1～4次の主な審査項目候補：**①経営力**
- 推論確度：**Medium**

「事業転換・垂直統合」は、団体旅行型の設備・運営を個人・インバウンド向け高付加価値型へ切り替える点を、利用可能な類型の中で最もよく表すため選んだ。垂直統合そのものが確認されたという意味ではない。

本件は、補正後の2次申請者中央値を8指標中6指標で下回る。それでも公開資料上は、実在する既存資産、複数ホテルの具体的な運営制約、設備と課題の対応関係、企業別の雇用・賃上げ目標、大規模な補助外負担が確認できる。これらが公開定量の弱さを補った可能性はあるが、需要・資金・地域波及の外部検証が弱いため、説明は**候補**にとどめる。
