## このノート集が答えられること

- どの公式指標で申請者と採択者の代表値に差があるか
- 公開PDFでどの指標をどの程度再現できるか
- 事業費・企業規模・指標重複が比較へ与える影響
- 数指標で劣っていても採択された公開事例が存在するか

## 答えられないこと

不採択企業の個票がないため、企業別の採択確率、指標1単位の因果効果、審査配点は推定できません。公式中央値は合格点ではありません。
