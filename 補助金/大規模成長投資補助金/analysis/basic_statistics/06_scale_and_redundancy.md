# 06 規模・事業費・指標重複

## 事業費との関係

同じ公募回内の順位相関です。

| 指標 | n | 事業費とのSpearman |
|---|---|---|
| 売上高CAGR | 99 | -0.141 |
| 労働生産性上昇率 | 375 | -0.093 |
| 1人当たり従業員給与上昇率 | 380 | -0.074 |
| 役員給与上昇率 | 312 | -0.024 |
| 売上増加額 | 338 | 0.467 |
| 付加価値増加額 | 372 | 0.483 |
| 給与総額増加額 | 372 | 0.390 |

- 売上・付加価値・給与の**増加額**は事業費と中程度に連動します。
- 売上CAGR、生産性率、1人当たり賃上げ率は事業費との関係が弱いです。
- 絶対額は事業費帯内比較または連続的な事業費調整を行い、率指標は原則として同回全体で比較します。

## 指標間の重複

| 指標A | 指標B | 共通n | Spearman |
|---|---|---|---|
| 全社売上高増加額 | 補助事業付加価値増加額（簡易補正推計） | 331 | 0.678 |
| 補助事業付加価値増加額（簡易補正推計） | 従業員給与支給総額の増加額（推計） | 371 | 0.670 |
| 年平均従業員目標賃上げ率 | 年平均役員目標賃上げ率 | 312 | 0.587 |
| 補助事業付加価値増加額（簡易補正推計） | 補助金額に対する付加価値増加額割合（推計） | 372 | 0.534 |
| 全社売上高増加額 | 従業員給与支給総額の増加額（推計） | 332 | 0.534 |
| 全社年平均売上高成長率 | 補助事業年平均労働生産性の伸び | 98 | 0.407 |
| 補助事業年平均労働生産性の伸び | 年平均従業員目標賃上げ率 | 375 | 0.366 |
| 全社年平均売上高成長率 | 全社売上高に対する投資額割合（近似） | 46 | 0.348 |

No.2・8・10は企業規模を共有し、No.8・14は同じ付加価値分子を共有します。9指標を単純に9票として数えると、同じ強弱を複数回数えます。

## 基礎属性の説明力

公募回を丸ごと外した低位度予測です。

| 属性ブロック | LOGO R² | LOGO Spearman |
|---|---|---|
| round_only | -0.022 | -0.121 |
| scale_only | 0.142 | 0.401 |
| industry_only | -0.036 | -0.039 |
| geography_only | 0.057 | 0.192 |
| structure_only | 0.028 | 0.082 |
| industry_geography | 0.048 | 0.200 |
| full_basic | 0.226 | 0.458 |

企業規模は一定の説明力を持ちますが、業種単独はほぼ説明せず、地域は小さい寄与にとどまります。属性調整は「優遇の証明」ではなく、公開数値が弱く見える構造の確認です。
