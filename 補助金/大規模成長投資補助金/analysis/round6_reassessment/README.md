# 採択者中央値を下回る指標が多い企業の再分析

このフォルダには、1～4次の公開企業PDFから作成した381の**企業レコード**を、現行の `public_metrics_dashboard.html` と同じ9指標で再分析した結果を収録しています。1～5次の公式代表値、第6次の事前公開資料、採択企業44社の目視評価も併せて使い、「複数指標で採択者中央値を下回る企業」を公開情報でどこまで説明できるかを検討しました。

381は申請案件数でも、公式の採択案件数でもありません。共同申請では一つの案件に複数企業が含まれるため、本分析の単位は一貫して「公開PDFから作成した企業レコード」としています。

最初に [round6_adoption_reassessment_report.html](round6_adoption_reassessment_report.html) を開いてください。数値の意味、ケース検索、第6次の申請設計まで一つの資料にまとめています。

## 主な結論

- 9指標のうち5指標以上を観測できた373レコード中、118（31.6%）が「観測指標の60%以上で、同じ公募回の採択者中央値を下回る」という分析上の条件に該当しました。主要品質フラグのないレコードに限ると58/199（29.1%）です。
- この118レコードのうち117には、申請者全体の代表値以上となる指標が少なくとも一つあります。80レコードでは観測指標の少なくとも半数が申請者代表値以上でした。ただし、厳密な過半数は67、丸め値との同値を除けば58です。
- 該当企業の数と顔ぶれは、指標構成によって大きく変わります。No.8を外すと101、No.13・14を外すと134、3領域に集約すると164です。「中央値を下回る企業」を、自然に定まる一つの集団として扱うことはできません。
- No.13（投資額／全社売上高）は、公式審査では値が高いほど評価上プラスです。ただし、過去5回の公式代表値では採択者側が申請者全体を上回ったのは1回だけでした。そのため、No.13を外した113/371（30.5%）という結果も、公式評価を否定する主分析ではなく感度分析として示しています。
- 採択企業44社の公開要約では、事業・工程の構造転換、能力制約、実行確度、政策との整合、地域・供給網への波及が頻繁に記載されていました。一方、採択企業どうしの40ペア比較では、公開指標が低い側だけに固有の差は、多重比較補正後には確認できませんでした。
- 第6次の申請では、過去の中央値を機械的に上回ることより、顧客需要、現状の能力制約、投資仕様、販売数量、売上、付加価値、賃金、地域波及、補助金の必要性を、同じ前提と数量で接続することが重要です。

## 再現方法

このフォルダで次を実行します。

```powershell
python analyze_round6_reassessment.py
python build_round6_reassessment_report.py
python validate_round6_reassessment.py
```

分析定義の詳細は [methodology.md](methodology.md) を参照してください。

## この分析の限界

- 非採択企業の個票と審査項目別得点がないため、採択確率、因果効果、審査配点は推定していません。
- No.8の1.1倍補正は、同じ1～4次の集計中央値を使った較正です。企業別精度の検証ではありません。公募回別交差検証も4個の公募回中央値だけを用いるため、企業順位や個社誤差を保証しません。
- Theil–Sen外挿は、5個の公式集計値に基づく探索的な参照値です。第6次の予測値や合格水準ではありません。
- 第6次に関する記述は事前公開資料に基づきます。正式公募の開始後は、[公式ダウンロードページ](https://chukentou-seichotoushi-hojo.jp/download/)で公募要領、申請様式、FAQとの差分を確認してください。
