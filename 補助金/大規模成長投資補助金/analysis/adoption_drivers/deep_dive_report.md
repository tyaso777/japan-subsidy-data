# どういう申請が通るのか――採択プロファイルの深掘り

## 1. 先に結論

このデータから最も妥当な理解は、**採択は単一指標の順位表ではなく、「明確な主戦場＋複数の補強材料＋実現可能性」のポートフォリオ評価**だということです。

公開PDF381社を同じ公募回内の百分位に直し、5つの定量軸へまとめると、次の類型に分かれました。

| 採択プロファイル | 社数 | 割合 | 解釈 |
|---|---:|---:|---|
| 企業変革投資型 | 82 | 21.5% | 企業規模に対して大きな投資を行う、または事業費が大きい |
| 複合バランス型 | 76 | 19.9% | 5軸中3軸以上が公募回内上位35% |
| 補助金効率型 | 62 | 16.3% | 付加価値、売上、給与、雇用の補助金当たり効果が強い |
| 成長・生産性型 | 57 | 15.0% | 売上成長、生産性、補助事業売上の成長が強い |
| 定性・非公開要因候補 | 57 | 15.0% | 公開5軸の最大値も公募回内55パーセンタイル未満 |
| 賃金・雇用型 | 29 | 7.6% | 賃上げ、給与総額、雇用増加が強い |
| 効果絶対額型 | 18 | 4.7% | 売上・付加価値・給与・雇用の絶対増加額が大きい |

全社の68.8%（262社）は、5軸のどれか一つで公募回内上位35%に入ります。19.9%（76社）は3軸以上で上位35%に入り、典型的な「総合力型」です。一方で31.2%（119社）は上位35%の軸が一つもありません。したがって、**強い定量軸は有力ですが、全採択案件に共通する必要条件ではありません**。

上位35%という閾値は公式基準ではなく類型化のための設定です。上位40%、30%、25%へ変えた結果は `threshold_sensitivity.csv` に保存し、特定の閾値だけに依存しないか確認できるようにしました。

## 2. 5つの定量軸

企業規模や公募回の違いを弱めるため、元の金額ではなく同じ公募回内の百分位を平均しました。

1. **成長・生産性**: 全社売上CAGR、全社売上増加額、労働生産性、取得できる場合は補助事業売上の割合・CAGR・増加額
2. **効果絶対額**: 売上、付加価値、給与総額、雇用の増加額
3. **補助金効率**: 付加価値、売上、給与総額、雇用の増加量を補助金額で除した値
4. **賃金・雇用**: 従業員賃上げ率、給与総額増加額、雇用増加数、補助金1億円当たり雇用増加数
5. **企業変革投資**: 投資額／基準売上高と事業費

この設計で重要なのは、絶対額と効率を別軸にした点です。例えば小規模案件は補助金効率が高くても効果絶対額が小さく、大規模案件は逆になるため、どちらか一方だけでは審査上の価値を表しません。

## 3. 既存指標で弱い企業は何で補っているか

前段の分析で、可視7指標の60%以上が採択者中央値未満となる企業を125社特定しました。

しかし、この125社を申請者全体の中央値と比べると、124社は少なくとも1指標で申請者中央値以上、90社は観測できる指標の半数以上で申請者中央値以上です。125社の「申請者中央値以上の指標割合」は平均59.7%でした。可視指標のすべてで申請者中央値を下回るのは1社だけです。

したがって、多くは「指標が弱いのに例外的に通った」のではなく、**採択者集団の中では下位でも、申請者集団の中では複数の競争力を持っていた**と読むのが正確です。

ただし、公式中央値は公募時の申請・採択集団、企業別PDFは主に交付決定後の公開集団です。特に1次・2次は母集団と公募回ラベルが一対一ではないため、個社の合否境界ではなく集計的な位置関係として使っています。

- 125社中66社は、5つの再構成軸のどれかで公募回内上位35%に入る
- 最も多い勝ち筋は、定性・非公開要因候補33社、企業変革投資型32社、補助金効率型29社
- 付加価値・補助事業売上の追加推計でも説明できなかった71社では、企業変革投資型が23社と最多
- この71社の平均百分位は、成長・生産性36.5%、効果絶対額30.3%、補助金効率32.8%、賃金・雇用35.5%に対し、企業変革投資だけ46.8%まで戻る

したがって、既存指標が弱い採択企業を理解する際、最初に見るべき追加情報は次の順です。

1. 補助金当たり付加価値
2. 売上規模に対する投資の大きさ
3. 補助事業そのものの市場成長・売上増加
4. 地域雇用と賃金への効果
5. 独自性、模倣困難性、市場根拠、実施可能性、加点

## 4. 「数値で上の会社があるのに、なぜ通るのか」

5軸がすべて計算できる企業について、同じ公募回内に「5軸すべてが同等以上で、少なくとも1軸が上」の別採択企業がいるかを確認しました。288社（75.6%）が、この意味で他の採択企業にパレート支配されています。

これは、75.6%の採択が不合理だという意味ではありません。むしろ次を示します。

- 採択は公開5軸を単調に足し上げたランキングではない
- 業種、市場、事業内容の違う案件を同じ物差しだけで順位付けしていない
- 独自性、地域性、実行体制、政策的重要性、プレゼンテーションが順位を変えている
- 公開PDFには、申請書と審査点の重要部分が収録されていない

特に「他社より全指標が低いのに採択」という見え方は、比較対象が採択企業だけであることにも注意が必要です。採択企業同士で優劣があっても、その企業が非採択企業より上だった可能性は残ります。

## 5. 文章から定性要因を復元できるか

公開PDFの文章を、公式審査項目に対応する5種類へ分けました。

- 経営力: 長期ビジョン、課題、戦略、収益、変革
- 先進性・市場性: 独自性、差別化、特許、競合、市場、需要、受注
- 地域波及: 地域雇用、地元調達、サプライチェーン、賃上げ、人材
- 実現可能性: 工程、許認可、資金、融資、体制、発注、契約、実績、リスク
- 政策適合: 省人化、DX、GX、脱炭素、国内回帰、安定供給、経済安全保障

単語数ではなく、該当語と数字・実績・受注・契約・比較等が同じ文にある「根拠文」の1,000文字当たり密度を使いました。

定量で説明できない71社は、全採択企業に比べて地域波及の公募回内百分位が53.1%、政策適合が51.4%とわずかに高い一方、先進性・市場性は47.1%、実現可能性は45.0%と低めでした。総合的な文章根拠密度が上位35%に入るものは11社だけです。

つまり、**公開2ページ程度の概要PDFからは、残る71社の採択理由を十分に復元できません**。文章が弱いのではなく、資金調達、工程、顧客証拠、競合比較、審査面接等が公開資料から落ちている可能性が高いと考えるべきです。

## 6. 通りやすい申請の構造仮説

公開データと公募要領を合わせると、申請は次の順で設計するのが合理的です。

### A. 必須条件を「満たす」から「審査しやすくする」へ

賃上げ、投資規模、事業期間等の要件を満たすだけでなく、基準値、目標値、期間、対象主体、計算式を一貫させます。主体や単位の不一致は、効果の信頼性を落とします。

### B. 一つの明確な主戦場を作る

最低でも次のどれかを、公募回上位を狙える水準で提示します。

- 補助金効率: 補助金1円当たりの付加価値、給与、雇用
- 効果絶対額: 付加価値、売上、雇用への大きな寄与
- 企業変革: 現在の売上規模に対して非連続な投資
- 成長性: 補助事業売上が市場成長率を上回る
- 賃金・地域雇用: 高い賃上げと地域への持続的波及

### C. 主戦場の分子と分母を両方示す

「付加価値が20億円増える」だけでなく、「補助金10億円に対し200%」と示します。逆に比率だけでなく絶対額も併記します。小規模高効率と大規模高効果の双方の弱点を防げます。

### D. 全社成長ではなく補助事業の因果鎖を示す

`設備・拠点 → 生産能力・品質 → 顧客・市場 → 補助事業売上 → 付加価値 → 賃金・雇用` の順で、数字を接続します。全社売上が伸びる計画だけでは、補助金による追加効果が曖昧です。

### E. 独自性を形容詞ではなく比較で示す

「革新的」「高付加価値」ではなく、競合方式との性能差、原価差、納期差、歩留まり、特許、認証、切替障壁、顧客評価を示します。市場成長率と自社計画を並べ、自社が市場を上回る根拠を説明します。

### F. 実現可能性を証拠で固める

発注先、見積、工期、許認可、土地、電力、人員、資金調達、顧客内示、試作実績、責任者、リスク対応を具体化します。高い目標値ほど、実現可能性の証拠が重要になります。

### G. 地域波及を一次効果と二次効果に分ける

- 一次効果: 新規雇用、賃上げ、地元採用
- 二次効果: 地元調達、物流、保守、取引先投資、供給網強靱化

人数や金額、対象企業数、期間を付け、単なる「地域貢献」で終わらせないことが重要です。

## 7. 申請時に持つべきスコアカード

| 領域 | 最低限の数値 | 強い申請に必要な証拠 |
|---|---|---|
| 効果絶対額 | 付加価値・売上・給与・雇用の増加額 | 設備能力から財務目標までの計算表 |
| 補助金効率 | 各増加額÷補助金額 | 比率と絶対額の併記、感度分析 |
| 事業成長 | 補助事業売上CAGR・増加額・全社比 | 市場調査、顧客内示、受注、価格・数量分解 |
| 企業変革 | 投資額÷直近売上高 | 投資前後の事業モデル・能力・収益構造 |
| 賃金・雇用 | 賃上げ率、給与総額、雇用増 | 職種、採用時期、定着、人材育成 |
| 独自性 | 性能・原価・納期等の競合差 | 特許、認証、実証、顧客評価、模倣障壁 |
| 地域波及 | 地元雇用・調達額・取引社数 | 具体的な地域企業、契約、波及経路 |
| 実現可能性 | 工程、資金、投資回収 | 見積、融資、許認可、責任体制、リスク対応 |

## 8. この分析で断定できないこと

本データには非採択企業の個票、申請書全文、審査項目別点数、加点、プレゼンテーション評価がありません。そのため、各指標が採択確率を何ポイント上げるか、審査員が実際に何を決め手にしたかは推定できません。

次に本当に「採択確率」を推定するには、少なくとも採択・非採択双方について、15指標、企業属性、加点、金融機関確認、申請文章、審査結果を同じ形式で集める必要があります。その段階で、交差検証付きのロジスティック回帰や勾配ブースティング、SHAP、業種・公募回固定効果を用いるのが適切です。
